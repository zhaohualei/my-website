## 二、各模块UI设计与功能说明

### 2.1 顶部Header

| 元素 | UI设计 | 功能 |
|------|--------|------|
| Logo | 深蓝底色 + 金色图标 + 公司名称 | 点击返回首页 |
| 语言切换 | 下拉框（中/英） | 切换网站语言版本 |
| 全站搜索 | 搜索图标 + 输入框 | 产品型号快速检索 |
| 联系热线 | 电话图标 + 号码 | 一键拨打（移动端） |

- 背景：`#0A3A6A`（科技深蓝）
- 强调色：`#D4AF37`（工业奢华金）
- 高度：60-70px，固定顶部

## 三、响应式设计方案

| 断点 | 布局调整 |
|------|----------|
| ≥1200px | 标准双栏布局（左侧导航240px + 右侧内容自适应） |
| 768-1199px | 左侧菜单收窄至图标模式（60px），悬停展开 |
| <768px | 顶部汉堡菜单 + 全屏内容，产品卡片改为全宽竖向排列 |

**移动端产品翻页适配：**
- 左右翻页按钮移至屏幕两侧边缘
- 支持手势滑动翻页
- 产品卡片改为上下结构（图片在上，信息在下）

## 六、技术实现建议

| 方面 | 建议方案 |
|------|----------|
| 前端框架 | Vue.js 3 / React 18（SPA单页应用） |
| UI组件库 | Element Plus / Ant Design |
| 样式方案 | SCSS + CSS Variables |
| 产品翻页 | Swiper.js / 自研轮播组件 |
| 图片展示 | PhotoSwipe / Viewer.js（灯箱效果） |
| 地图集成 | 百度地图 API / 高德地图 API |
| 表单处理 | 邮件通知 + 后台管理系统 |
